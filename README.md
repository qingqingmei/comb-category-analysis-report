# 梳子品类评价分析报告

这是一个 GitHub Pages 静态站点。`index.html` 已包含完整报告、下钻数据和分析助手，不依赖外部服务。

## 发布方式

1. 在 `qingqingmei` 组织下新建一个 GitHub 仓库，例如 `comb-category-analysis-report`。
2. 上传本目录下的 `index.html`、`.nojekyll` 和 `README.md` 到仓库根目录。
3. 在仓库 `Settings -> Pages` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. 保存后等待 GitHub Pages 生成公网链接。

发布后的访问链接应为：

```text
https://qingqingmei.github.io/comb-category-analysis-report/
```

如果使用其他仓库名，则链接为：

```text
https://qingqingmei.github.io/你的仓库名/
```
